üìä Upload CSV Dataset
Upload one or more CSV files containing labeled email data for training.

CSV Format Requirements:

  • subject - Email subject line (optional, can be empty)
  • body - Email body content (required)
  • label - Classification label (required)
Example CSV format:

subject,body,label
"URGENT Account Alert","Click here to verify...",1
"Meeting Tomorrow","Hi team, reminder about...",0
"You won $1000!","Congratulations! Click to claim...",1
"Project Update","The quarterly report is...",0

Label Format:

The label column accepts multiple formats:

  • Numeric: 0 = legitimate, 1 = phishing
  • Text (phishing): "spam", "phishing", "malicious", "bad"
  • Text (legitimate): "ham", "legitimate", "safe", "good"
You can select multiple CSV files at once

üí° Training Tips:

  • Use at least 100 examples of each class (phishing and legitimate)
  • Ensure balanced dataset - equal numbers of phishing and legitimate emails
  • More diverse examples lead to better accuracy
  • Training may take several minutes depending on data size
  • The model will automatically save after successful training
  • If you don't have a dataset, use the fix_dataset.py script to create a test dataset
